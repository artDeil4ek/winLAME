Binaries

The binary versions of winLAME run on the following operating systems: Windows 7, Windows Vista, Windows 8, Windows 8.1 and Windows 10.

Setup package (recommended!)

The setup is provided as a Windows Installer .msi file and can be installed by double-clicking the downloaded file.

winLAME-2017-beta2.msi (5.9 MB)
Download from Sourceforge.net

winLAME Portable Edition

The Portable Edition of winLAME can be used to run winLAME from USB sticks without the need to install the application. Created with PortableApps.com.

winLAMEPortable_2017_beta_2_English.paf.exe (3.8 MB)
Download from Sourceforge.net

Standalone zip archive

The zip archive contains all files needed to run winLAME without installing it.

winLAME-2017-beta2.zip (19.2 MB)
Download from Sourceforge.net

Source code

winLAME is an Open Source project, and so naturally also provides the whole source code of winLAME.

winLAME-2017-beta2-source.zip (15.2 MB)
Download from Sourceforge.net

You can also browse the winLAME source code by using Sourceforge's Code Browser
