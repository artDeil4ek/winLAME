Preset Selection Page

This is the preset selection pages. It only appears when the presets settings file (presets.xml) is installed, and when MSXML3 is available. Normally it is installed with Internet Explorer 5 or higher. Also, presets for the specific output module must be available in the presets settings file to show this page.

Preset List: lists all available presets for the currently selected output module (from the output page). The description box shows the description for the currently selected preset. Double-clicking on one preset shows the settings parameter that are used for that preset.

Preset Description Box: presents a description of the currently selected preset.

Possible next pages: LAME Settings, Ogg Vorbis Settings, Wave Output Settings, AAC Settings, Encode.
