Encode Page

This is the encode page where the encoding is done.

Encode Control

Buttons:

Start encoding
Starts encoding; when encoding was stopped before, winLAME starts with the first file not fully processed.
Pause encoding
Pauses encoding; no processor time will be used during pausing
Stops encoding
Stops encoding; when pressing the start button again, the currently stopped file will be encoded again
Sends winLAME to the system notification area
Sends winLAME to the system notification area. The encoding status is visible with the color of the icon in the notification area; a red icon means that winLAME is currently encoding, a green icon means that winLAME is not encoding or has finished all its encoding jobs.

Note: When all files are encoded, you can use the back button to set up another encoding process.

Progress Information

Shows info about the number of files already encoded, the currently processed file and a progress meter. the remaining time is estimated from the time already used to encode the file.

Encoder Setup Information

Shows info about the current parameters used for the reading and encoding process.
