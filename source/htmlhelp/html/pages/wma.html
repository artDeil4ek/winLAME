Windows Media Audio Settings Page

This is the output settings page that configures Windows Media Audio encoding.

Constant bitrate

Specifies a constant bitrate in kbit/s that should be used for encoding.

Quality (VBR)

Specifies a qualit value in percent. Allowed values range from 10% to 100%.

Possible next pages: Encode Page (Classic UI) or Finish Page (Modern UI).
