CD Audio Extraction Page

Classic UI only

This is the page where CD Audio is read from the CD, in order to encode it in the next step.

Start

The Start button starts extraction of the audio tracks. The progress bar on the left side shows the extraction progress.

Track List

The track list shows all audio tracks to be extracted, and the progress of extraction.

Autostart encoding after finished extraction

When checked, proceeds to the next page and starts encoding right after that.
