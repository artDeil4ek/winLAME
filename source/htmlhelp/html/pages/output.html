Output Settings Page

This is the output settings page that configures general output settings.

Output Module: lets you select the output module that winLAME should use to encode all of your files. Some modules are only available when the proper DLL files are placed into the winLAME folder.

Output Location: lets you specify the output folder. The button at the left lets you select the folder. The folder history is available by clicking the down arrow.

use input file's folder as output location: uses the same output location as the input file has, for each input file. This allows multi-folder encodes.

delete after encoding: indicates if the input file should be deleted after successful encoding of that file.

overwrite existing files: indicates if already existing files will be overwritten

after encoding, do this: when checked, performs one of the following actions when all files are encoded. Not all actions are available on all Windows versions.
Close winLAMEcloses winLAME when finished
Shutdownshuts down Windows when finished
Log offlogs off the current user when finished
Hibernatewrites RAM contents to hard disk and powers down
Suspendsuspends the system to RAM and powers down

create output playlist: when checked, a m3u playlist with the name in the edit field is created while encoding files. Automatically enabled when a playlist was added or dropped into the input files list.
