winLAME Wizard Pages

winLAME uses a wizard-dialog style appearance (looks like an Install Shield Wizard), to configure all details of encoding or decoding.

The following configuration wizard pages can appear in winLAME :

  • Input Files
  • Output Settings
  • Preset Selection
  • LAME Settings
  • Ogg Vorbis Settings
  • Wave Output Settings
  • AAC Settings
  • Encode

Note: in winLAME, press F1 to get directly to the help page of the current wizard page. There are tool tips for every control on a page that show up when one hovers over a control longer than one second.

System Menu

uninstall winLAME: quits winLAME and starts the winLAME uninstaller.

about winLAME: shows the winLAME about box. The about box lists informations about winLAME and the versions of the used libraries.
